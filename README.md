# Accounting CRM Platform

This project is a CRM system designed for accounting firms to manage clients, tasks, documents, and internal operations efficiently.

## 🚀 Technologies

- Frontend: React (JavaScript)
- Backend: ASP.NET Core Web API (C#)
- Database: SQL Server
- Architecture: Layered Architecture (API, Application, Domain, Infrastructure)

## 📂 Project Structure

- `pycrm/` → Frontend (React application)
- `WebApi-Accounting/` → Backend (ASP.NET Core API)
- `database/` → SQL scripts and database structure

## 🎯 Features

- Client management
- Task tracking system
- Document management
- User and role management
- Event tracking
- Dashboard and reporting

## ⚙️ How to Run

### Backend

1. Open the solution in Visual Studio
2. Configure your database connection manually
3. Run the API project

### Frontend

cd pycrm
